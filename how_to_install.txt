How to install nispat?

1) install Anaconda3
2) install pip with  "conda install pip"

optional, but recommended:
    a) create enviornment with " conda create --name normative_modeling " 
    b) activate environment by " source activate normative_modeling " 


3) download nispat from github
4) unzip nispat into download folder 
5) change dir to download folder using the terminal and execute " pip install nispat-master/ "

or

3) clone nispat from github
4) change dir to download folder using the terminal and execute " pip install nispat/ "

You should be all set now!
